# GuessGame2

這是從 macOS App 移植到 iOS 的猜數字遊戲，使用者猜測四個不重複數字，並提示 AB 值為線索。

> macOS App 專案網址 [https://github.com/kaichingchang/GuessGame](https://github.com/kaichingchang/GuessGame)
